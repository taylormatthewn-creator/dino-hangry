# Dino Hangry Rocks

A kid-friendly Streamlit game prototype: a T-Rex stomps rocks to find food, unlocks a ninja helper around level 6, solves crystal mini-games between levels, and battles cavemen bosses every 10 levels.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub + Streamlit Community Cloud

1. Create a new GitHub repository.
2. Upload `app.py`, `requirements.txt`, and this `README.md`.
3. Go to Streamlit Community Cloud.
4. Connect your GitHub repo.
5. Set the main file path to `app.py`.
6. Deploy.

## Notes

This is a first playable prototype. It uses buttons, simple stats, emoji art, and Streamlit session state so it is easy to edit with kids.
